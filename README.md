# HRTOS_Mini
HRTOS_Mini for 8051

### 系统说明
本系统是HRTOS的一个精简版本，本系统可以在任意一款8051处理器上运行。
迷你HRTOS是一款免费、开源的操作系统。
为了让您更快、更容易的上手，我们舍去了所有的汇编部分，全部用C51编写，让您更快、更自然的上手。
如果你对HRTOS精简版有改进意见，或者愿意为了一个更好的系统而作出自己的奉献。
